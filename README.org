** etuk-template

A Leiningen template for creating simple automation with etuk

*** Basic Usage

#+BEGIN_SRC sh
lein new etuk awesome-automate
cd awesome-automate

# If you using Leiningen
lein deps

# If you prefer Boot
boot build
boot
#+END_SRC
